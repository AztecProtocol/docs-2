# Language Description

Comprehensive guide to the Aztec.nr language and its features.

## Overview

Aztec.nr is a domain-specific language for writing smart contracts on Aztec. It's based on Noir, a language for writing zero-knowledge circuits, with additional features specific to Aztec smart contracts.
